# Project: Wrangle and Analyze Data

The seventh project of [Udacity's](https://www.udacity.com) **Data Analyst Nanodegree**

Using Python and its libraries to gather, assess, and clean data from various source. The assessment steps includes identifying data quality and tidiness issues.

## Project Files
* `wrangle_act.ipynb`: code for gathering, assessing, cleaning, analyzing, and visualizing data
* `act_report.pdf`: documentation of analysis and insights into final data
* `wrangle_report.pdf`: documentation for data wrangling steps: gather, assess, and clean
* `twitter_archive_enhanced.csv`: enhanced WeRateDogs Twitter archive
* `image_predictions.tsv`: image classification of the breeds of dogs
* `tweet_json.txt`: data gathered via Twitter API
* `twitter_archive_master.csv`: combined and cleaned data
